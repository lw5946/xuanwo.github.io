POJ 1159 Palindrome

2014-08-12
Exercise
Comments

题目

源地址：

http://poj.org/problem?id=1159

理解

最关键的公式是：

if (str[i] == str[j])
    DP[k][j] = DP[1 - k][j - 1];
else
    DP[k][j] = min(DP[1 - k][j], DP[k][j - 1]) + 1;

注意到k空间的循环利用，节省了空间，感谢大牛提供的思路。
